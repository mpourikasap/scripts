# Χωρική Ανάλυση Προσβασιμότητας με χρήση Python & QGIS

Αυτό το project υλοποιεί μια χωρική ανάλυση καταλληλότητας και προσβασιμότητας εδάφους στο *Ποικίλο Όρος*, χρησιμοποιώντας το *QGIS* και την *Python*.

# Επισκόπηση
Η ανάλυση αυτοματοποιεί τη διαδικασία ταξινόμησης των κλίσεων του εδάφους από ένα Ψηφιακό Μοντέλο Υψομέτρου (DEM) σε τρεις κατηγορίες προσβασιμότητας:
- Υψηλή (0-15°): Εδάφη με εύκολη πρόσβαση.
- Μέση (15-30°): Εδάφη μέτριας δυσκολίας.
- Χαμηλή (>30°): Απότομα εδάφη / περιορισμένη πρόσβαση.

# Τεχνολογίες & Εργαλεία
- QGIS 3.40.15
- Python 3.11 (PyQGIS API)
- GDAL/OGR για την επεξεργασία raster δεδομένων
